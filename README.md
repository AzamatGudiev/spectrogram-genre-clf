# Классификация музыкальных жанров по спектрограммам

Сравниваю три подхода к задаче — от простой полносвязной сети до предобученной модели. Датасет GTZAN, 10 жанров.

## Что сравнивал

| Модель | Accuracy |
|--------|----------|
| MLP (baseline) | 30.46% |
| EfficientNet B0 | 92.67% |
| CNN (своя) | 100%* |

*CNN переобучилась на маленьком датасете — val_loss уходит в ноль уже на второй эпохе. EfficientNet в этом плане честнее.

## Жанры

blues, classical, country, disco, hiphop, jazz, metal, pop, reggae, rock

## Как запустить

Нужен датасет GTZAN (спектрограммы). Кладёшь его в `data/gtzan_spectrograms/images_original/` и запускаешь ноутбук по порядку — там всё по секциям.

Обучал на M2 (mps), на CPU тоже работает, просто дольше.
