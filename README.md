# Классификация музыкальных жанров по спектрограммам

Сравниваю три подхода к задаче — от простой полносвязной сети до предобученной модели. Датасет GTZAN, 10 жанров, 999 реальных спектрограмм.

## Результаты

| Модель | Accuracy (test) |
|--------|----------------|
| MLP (baseline) | 30.46% |
| CNN (своя сеть) | 63.09% |
| EfficientNet B0 (Transfer Learning) | 63.76% |

CNN показывает выраженное переобучение (train ~97%, val ~59%) — при 701 обучающем примере и 8.5 млн параметров это ожидаемо. EfficientNet B0 с замороженным backbone и всего 12 810 обучаемыми параметрами даёт чуть лучший результат с меньшим переобучением (train 65.6%, val 57.1%).

## Что сравнивал

1. **MLP** — полносвязная сеть (baseline из курса)
2. **CNN** — своя свёрточная сеть: 3 блока Conv→ReLU→MaxPool + FC
3. **EfficientNet B0** — transfer learning с предобученными весами ImageNet

## Жанры

blues, classical, country, disco, hiphop, jazz, metal, pop, reggae, rock

## Как запустить

Нужен датасет GTZAN (спектрограммы). Кладёшь в `data/gtzan_spectrograms/images_original_real/` (по одной папке на жанр) и запускаешь ноутбук по порядку.

Обучал на M2 (mps), на CPU тоже работает, просто дольше (~30 мин вместо 15).
